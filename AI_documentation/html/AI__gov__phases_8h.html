◆ AI_gov_sysfs_load_profile()

signed int AI_gov_sysfs_load_profile ( enum PHASE_ENUM  new_phase)

Unloads the current phase and it's profile and loads a new phase and profile.

Firstly the old profile's sysfs entry (kobject) is removed from the sysfs heirarchy followed by the changing the phase within the AI_gov struct. This phase's profile is then actualised by loading the respective profile.

Parameters
new_phase- The new phase that is to be loaded, must be an already registered phase
Returns
0 on success

◆ AI_phases_set_defaults()

unsigned char AI_phases_set_defaults ( void  )

Sets phase profile attribute default values.

All default attributes must be set within this function. To add profile defaults the temp phase_profile pointer must first be set to the profile to be set. This can be done using AI_phases_get_name. The macro GET_ATTRIBUTES_W_PROFILE can be used to retrieve the attributes of the current phase through typecasting the attributes void pointer. The attributes can thusly be set to the default values. As all phase profiles are calloc'd initially, all default values are 0 unless set within this function

Returns
0 on success
