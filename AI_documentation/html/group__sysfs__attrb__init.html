Detailed Description

Each phase must have a macro defined that specifies each attribute that is to be added to the phase's sysfs profile folder. The macro must have the form of:
SYSFS_::phase name::_ATTRIBS(ATTRB)
ATTRB(#phase name#, #attribute 1 name#) \
ATTRB(#phase name#, #attribute 2 name#) \
etc...
This is because the macro creates the initilisation code for each specified attribute and connects it with the specified phase. Each phase's attibutes must be in a single macro, you cannot mix the phase within a single macro.
